# Aplikasi Laporan Data Barang
Aplikasi data barang menggunakan Hibernate, JasperReport, dan Java

## Fitur
- Fitur yang tersedia adalah berupa GUI untuk menyimpan, update, menghapus, dan mencetak laporan data barang
- Koneksi database menggunakan Mysql, dan Hibernate

## Teknologi
- Java Netbeans 8.2
- Hibernate
- JasperReport
- Database MySQL

## Database
Database MySQL terdiri dari tabel :
- Barang

## Setup
- Buat database beserta tabelnya pada MySQL
- Buat GUI beserta fiturnya pada Java Swing
- Import library Hibernate dan JasperReport
